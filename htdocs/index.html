Web Designer vs Web Programmer

WebデザイナーとWebプログラマについて

Webデザイナー

Webデザイナーは、Webサイトの目的・コンセプトにもとづいてデザイン をイメージし、実際にデザインとして形にしていく職種です。Web デザイナーになるには、専門学校などでWebデザインやコーディング について学び、Web制作会社で働くのが一般的です。そのうえで、Web デザイナーはデザインセンスを磨いたり、アクセシビリティやユーザ ビリティに配慮した「UI（表示される画面）」を実装したりする技術 が求められます。Webデザイナーは自分でデザインを考えたり作ったり することが好きな人におすすめです。

Webプログラマー

Webプログラマーは、Webサイトの裏側で動くさまざまなシステムに ついて、プログラミング言語を用いて開発します。Webプログラマーは、 システム開発のためのプログラミングスキルが求められるため、開発に使う 言語を修得する必要があります。とくに最近はPHP、Ruby、Pythonが多く 用いられているため、これらのうち1つ以上で開発ができるレベルに達する ことが求められます。Webプログラマーは、システムや仕組みを考えたり、 プログラムを作成するのが好きな人が向いている職種といえます。

Advantages of Studying Web

Webを学ぶメリット

驚異的な需要

Web開発者の需要は増え続けています。企業は常に良いWeb開発者やデザイナー を探しており、需要は将来的に成長する予定です。

仕事の柔軟性

フリーランスとして、パートタイム、フルタイム、企業、政府、自分のために、自宅から、あるいはバハマでの休暇中に働くことができます！

給料がいい

数時間の作業で高額な報酬が得られることも多く、非常にやりがいのある仕事であり、キャリアの選択肢として非常に人気があります。

資格

ウェブ開発者になるには、必ずしも高度なコンピュータサイエンスの学位が必要なわけではない。知識とポートフォリオに基づいて採用される。

クリエイティブ

ウェブデザインは、コンピュータを使って美しく機能的なものを作ることができるため、クリエイティブなタイプの人には最適です。

Personal Opinion About Courses

WEBデザイン・WEB開発の授業の感想

Webサイトプロトタイプ演習

Webサイトプロトタイプ演習では、ウェブサイトの作り方の基本を学びましたが、講師の方がとても親身に生徒のことを考えてくれます。当時、私はウェブサイトを扱った経験があまりありませんでした。ウェブサイトの作り方の基本を熱心に教えていただきました。基礎がしっかりできたことで、自分でも作りたいと思うようになりました。

Webサイト制作実践演習

Webサイト制作実践演習は、前学期に受けたデザインとプログラミングの両方の授業が必要なので、少し高度な内容になっています。しかし、講師はDHUの中でも最も親切で理解ある講師の一人です。上級クラスなので、課題はそれほど簡単ではありませんが、自分のスキルをさらに向上させるために、自分を追い込みたいと思うようになりました。

コンテンツデザインゼミ

ゼミは、卒業研究に取り組むための必須条件です。とはいえ、DHU在籍中は1つしか選べません。残念ながら私はこのゼミを選択しませんでしたが、それは内容ではなく、ゲームゼミを追求したかったからです。でも、他のゼミの先生と同じように、このゼミでも卒業制作を最高のものにするために、できる限り指導してくれると思います。
